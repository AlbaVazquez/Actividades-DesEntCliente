# JobQuest  - Gestor de Candidaturas

## Descripción de la Aplicación
**JobQuest** es una aplicación web (SPA) diseñada para ayudar a los desarrolladores y profesionales a organizar y hacer seguimiento de sus candidaturas de empleo. 

Su propósito principal es ofrecer un panel de control intuitivo donde el usuario puede añadir nuevas ofertas a las que ha aplicado, establecer su estado (Enviado, Entrevista, Oferta, Rechazado) y eliminarlas cuando ya no sean necesarias. Todos los datos persisten en la nube en tiempo real.

## Tecnologías y Estilos
Para el desarrollo de esta aplicación se han utilizado las siguientes tecnologías:
* **Core:** React 18, TypeScript y Vite.
* **Base de Datos:** Firebase Firestore (Cloud Database) para el CRUD completo.
* **Estilos:** Se ha elegido **Bootstrap 5** por su robusto sistema de rejilla (Grid) y sus clases de utilidad predefinidas. Esto ha permitido crear una interfaz 100% *responsive*, limpia y profesional de manera rápida, utilizando componentes como `Cards`, `Forms` y `Badges` dinámicos según el estado de la candidatura.
